# Management Portal

A web-based management portal built with ASP.NET, C#, .NET Framework, and MySQL.

## Features
- User authentication and roles
- CRUD operations on database
- Admin dashboard
- Reporting tools

## Tech Stack
- ASP.NET MVC / C#
- MySQL database
- Visual Studio IDE

## Run Locally
1. Open `ManagementPortal.sln` in Visual Studio
2. Restore NuGet packages
3. Set startup project and run
