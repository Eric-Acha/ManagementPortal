# Management Portal

**Web-based Management Portal built with ASP.NET, C#, .NET Framework, and MySQL.**

Management Portal is a robust web application designed to streamline administrative tasks for organizations. It provides user authentication, role-based access, CRUD operations, reporting, and integration with a MySQL database. The app includes a structured front-end using HTML, CSS, and JavaScript, along with a responsive admin dashboard.

---

## Features

- **User Authentication:** Secure login system with role-based access control.  
- **CRUD Operations:** Manage data for employees, products, or any organizational records.  
- **Admin Dashboard:** View summaries, reports, and system statistics.  
- **Database Integration:** MySQL backend for reliable data storage.  
- **Responsive UI:** Clean front-end using HTML, CSS, and JavaScript.  

---

## Tech Stack

- **Backend:** ASP.NET MVC, C#  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** MySQL  
- **IDE:** Visual Studio  
- **Framework:** .NET Framework
